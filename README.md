# PYT Selector

PYT Selector는 스포츠 카드 브레이크 참여자를 위한  
**PYT 팀 추천 웹사이트**입니다.

사용자가 팀별 가격, 주요 선수, 기대값, 가성비를 비교하고  
자신에게 맞는 팀을 선택할 수 있도록 돕는 것을 목표로 합니다.

## Tech Stack

### Frontend
- Next.js 16
- TypeScript
- Tailwind CSS
- Node.js 24.x LTS

### Backend
- Java 19
- Spring Boot 4.0.5
- Gradle
- Spring Data JPA
- QueryDSL
- Spring Security
- MariaDB
- Lombok

## Development Environment

### Frontend
- Node.js: 24.x LTS
- npm: latest stable

### Backend
- Java: 19
- Spring Boot: 4.0.5
- Gradle: wrapper version
